كيف تعمل الخدمة؟

من تسجيل الحساب إلى استقبال أول كود تحقق لدى عميلك – 3 خطوات بسيطة.

1

أنشئ حسابًا واحصل على API Key

تسجل في We Pioners Auth، فعّل رقم الواتساب الخاص بك، واحصل على مفتاح API السري الخاص بتطبيقك.

2

اربطه بتطبيقك أو موقعك

أضف استدعاء بسيط من Backend لديك لإرسال كود التحقق عند تسجيل المستخدم، تسجيل الدخول، أو أي عملية حساسة.

3

تحقق من الكود بكل سهولة

استخدم Endpoint مخصص لتأكيد الكود الذي أدخله المستخدم، واسمح له بإكمال التسجيل أو العملية بأمان.

أمثلة كود للمطورين

أمثلة سريعة يمكنك نسخها ولصقها في مشروعك مباشرة.

Node.js (Axios)

import axios from "axios";

const apiKey = "YOUR_API_KEY";

async function sendOtp(phone) {
  const res = await axios.post(
    "https://api.wepioners-auth.com/v1/whatsapp/otp",
    {
      phone,
      channel: "whatsapp",
      code_length: 6,
      expire_in: 300
    },
    {
      headers: {
        Authorization: `Bearer ${apiKey}`,
      },
    }
  );

  console.log(res.data);
}

PHP (cURL)

<?php
$ch = curl_init("https://api.wepioners-auth.com/v1/whatsapp/otp");

$data = [
  "phone" => "+9647700000000",
  "channel" => "whatsapp",
  "code_length" => 6,
  "expire_in" => 300
];

curl_setopt_array($ch, [
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_POST => true,
  CURLOPT_POSTFIELDS => json_encode($data),
  CURLOPT_HTTPHEADER => [
    "Content-Type: application/json",
    "Authorization: Bearer YOUR_API_KEY"
  ]
]);

$response = curl_exec($ch);
curl_close($ch);

echo $response;

ندعم التحويل البنكي، بوابات دفع إلكترونية، ويمكن إضافة طرق دفع أخرى حسب بلدك واحتياجك.
